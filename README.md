# Hydra Q FincaOS V24

Streamlit app connected to Google Sheets public CSV.

Files required in the GitHub repo root:
- app.py
- requirements.txt
- README.md
- hydraq_logo.png
- cama_6_y_3.jpeg
- vista_sur.jpeg
- vista_este.jpeg

Optional support file:
- FincaOS_Data_V24_Clean.xlsx

Changes in V24:
- Tree cards split into 3 sections: general information, registered events, and future actions + notes.
- Registered events remain as an embedded chronological table inside each tree card.
- Future actions section now shows suggested next soil fertilization, pending actions, and notes.
- Clean starter database included with GPT / Apps Script test events removed from EventosAgricolas and LogCambiosGPT.
- Added a starter sheet: GuiaAbonadoArboles.
